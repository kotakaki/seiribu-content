## キャプション候補

### 3つの基本手順

売る・自力で捨てる手順を挟むことで、業者への依頼費用を最小化できます。

### 4つの方法比較

それぞれの方法の「費用」と「手間」のバランスを知り、状況に合わせて使い分けましょう。

### 境界線チェックリスト

無理をして自力で進めると、かえって危険や出費を伴うケースがあります。
