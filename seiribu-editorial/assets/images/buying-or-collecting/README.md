## 設置位置・キャプション

### `buying-or-collecting-eyecatch.png`

- 設置位置: アイキャッチ合成用素材
- キャプション候補: なし
- ロゴ: none

### `buying-or-collecting-inline-flow.png`

- 設置位置: 本文中のCMSブリーフ位置
- キャプション候補: 不用品回収は「払う」、買取は「もらう」という正反対のサービスです。
- ロゴ: white-backed:右下; model-native-text

### `buying-or-collecting-photo.png`

- 設置位置: 本文中のCMSブリーフ位置
- キャプション候補: 専門知識がないと、査定対象になる物まで処分対象として扱われてしまうことも。
- ロゴ: white-backed:右上

### `buying-or-collecting-inline-three-boxes.png`

- 設置位置: 本文中のCMSブリーフ位置
- キャプション候補: 古いから価値がないと自己判断せず、まずは査定対象になるか確認しましょう。
- ロゴ: white-backed:右上

### `buying-or-collecting-inline-steps.png`

- 設置位置: 本文中のCMSブリーフ位置
- キャプション候補: 残すものを分けた後、買取で価値を確認してから処分を進めると、見落としを減らしやすくなります。
- ロゴ: white-backed:左上; model-native-text

## 運用メモ

- アイキャッチ素材はCanvaでタイトル、サブタイトル、ロゴを手動仕上げする
- 本文画像・図解はブランド帰属表示としてセイリ部ロゴを合成済み
- WordPressアップロード後、記事本文内のCMSブリーフは削除する
