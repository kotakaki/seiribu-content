## ロゴ配置

- 2枚とも右下の余白にセイリ部ロゴを配置済み
- ロゴ元画像は `../brand/seiribu-logo.png`

## 制作メモ

- 本記事の画像は日本語ラベルの正確さが重要なため、画像生成AIに文字入り図解を丸投げせず、レイアウト制御された図解として使う
- 宗教的な表現、供養、お焚き上げ、寺社、仏壇の描写は画像に入れない
- 不用品回収業者だけを強く推す広告表現にしない

## キャプション候補

### 買取不可になる理由

買取不可の理由は、品物の価値そのものではなく、再販売できる状態かどうかで判断されることが多いです。

### 手放す3つの選択肢

売れなかった不用品は、費用を抑えるか、時間を短縮するかで手放し方を選びましょう。
