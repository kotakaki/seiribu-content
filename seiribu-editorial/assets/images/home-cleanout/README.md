## 使い方

- アイキャッチは `home-cleanout-eyecatch-branded.png` を固定ページのアイキャッチ画像に設定する
- `home-cleanout-eyecatch.png` は旧版または比較用として残す
- 記事内画像は「売る・残す・処分するの判断基準」の見出し直下に入れる
- `home-cleanout-inline-principles.png` は「実家の片付けで挫折しないための3つの鉄則」の見出し直下に入れる
- `home-cleanout-inline-steps.png` は「実家の片付けはどこから始める？効率的な順番」の導入直後に入れる
- `home-cleanout-inline-three-boxes.png` は「ステップ4：リビング・収納の奥・親の趣味品へ進む」の本文中に入れる
- 記事内画像・図解には、右下または左上の余白にセイリ部ロゴを小さく配置する
- ロゴは見出し、人物の顔、重要な品物を避けて配置する

## キャプション候補

### アイキャッチ

実家の整理は、いきなり捨てるのではなく、売る・残す・処分するに分けることから始めます。

### アイキャッチ（ブランド型）

実家の片付けは、家族で確認しながら「売る・残す・処分する」の順番で進めると迷いにくくなります。

### 記事内画像

迷う荷物は、まず「売る」「残す」「処分する」の3つに分けて考えると整理しやすくなります。

### 3つの鉄則

実家の片付けは、勝手に捨てず、一人で抱え込まず、次の行動別に分けることから始めます。

### 4つのステップ

話し合いから始め、判断しやすい場所を片付けてから、思い出の品や趣味品に進みましょう。

### 3つの箱

迷う荷物はその場で処分せず、売れる可能性、家族確認、処分方法の3つに分けて整理します。
