## キャプション候補

### H2直下イメージ

実家の片付けでは、古い物をすぐに捨てず、品名や付属品を確認してから判断します。

### 記事内図解

迷う荷物は「売れる可能性がある物」「確認してから判断する物」「処分方法を確認する物」に分けると整理しやすくなります。

## 生成プロンプト

### H2直下イメージ

実家の部屋で、段ボールに入った古い本、フィルムカメラ、レコード、ブランドバッグ、切手帳、茶道具などを、親子（40代から50代の子と70代の親）が一緒に確認している様子。明るく整理された雰囲気。2Dの日本向け編集イラスト調。ロゴ、価格表示、現金、文字は入れない。

### 記事内図解

3つの箱に荷物を分けるフロー図。左から「売れる可能性がある物」（緑）、「確認してから判断する物」（黄）、「処分方法を確認する物」（グレー）。各箱の下に代表的な品物の小さなイラスト。シンプルで見やすいインフォグラフィック。
