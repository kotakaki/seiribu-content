### H2直下イメージ

実家の部屋で、段ボールに入った古い本、フィルムカメラ、レコード、ブランドバッグ、切手帳、茶道具などを、親子（40代から50代の子と70代の親）が一緒に確認している様子。明るく整理された雰囲気。2Dの日本向け編集イラスト調。ロゴ、価格表示、現金、文字は入れない。
