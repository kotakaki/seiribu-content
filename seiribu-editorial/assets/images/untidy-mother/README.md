## キャプション候補

### 片付けられない心理

逆ギレの裏には、母親なりの葛藤と不安が隠れています。

### 第三者の介入

親子だけで抱え込まず、第三者を挟んで落ち着いて相談する選択肢もあります。

## 制作メモ

- 記事ブリーフに基づき、画像生成素材を一時置き場で作成した
- アイキャッチはクロマキー素材から背景を抜き、ローカル合成でタイトルとセイリ部ロゴを配置した
- 本文画像は生成AIに文字やロゴを描かせず、左上にセイリ部ロゴを小さくローカル合成した
- 汚部屋、暗い遺品整理、高額査定広告の印象を避け、明るく清潔な日本の実家らしさを優先した
