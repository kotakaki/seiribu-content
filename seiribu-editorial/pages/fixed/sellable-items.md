## 実家の片付けでは、捨てる前に価値確認をする

実家の片付けでは、古く見える物や使っていない物をまとめて処分したくなることがあります。

しかし、親の家には、家族が価値を知らないまま保管されている物もあります。古い本、ブランド品、カメラ、レコード、切手、古銭、趣味品、コレクションなどは、状態や種類によって査定対象になることがあります。

「使わないから捨てる」と決める前に、売れる可能性があるかを確認しましょう。

## 捨てる前に確認したい売れるもの

| 品物 | 確認ポイント | 向いている売り方 |
| --- | --- | --- |
| 本、専門書、全集 | ISBN、書き込み、分野、発行年 | 古本買取、専門店 |
| ブランド品 | ブランド名、状態、付属品、保証書 | ブランド買取 |
| カメラ、レンズ | メーカー、型番、動作、カビ | カメラ専門買取 |
| レコード、CD | ジャンル、盤面、帯、限定盤 | 専門店、フリマ |
| 切手、古銭、記念硬貨 | 種類、状態、保管方法 | 専門買取 |
| フィギュア、おもちゃ | 箱、付属品、シリーズ名 | ホビー買取 |
| 時計、アクセサリー | ブランド、素材、動作 | 貴金属・ブランド買取 |
| 工具、楽器 | メーカー、型番、動作 | 専門買取 |

## 売れやすいものと売れにくいものの違い

売れやすいものには、いくつか共通点があります。

- 商品名や型番がわかる
- 付属品や箱が残っている
- 専門店が扱っているジャンルである
- 壊れていても部品や素材に価値がある
- コレクター需要がある

一方で、状態が悪いもの、需要が少ないもの、衛生面で再利用しにくいものは、買取が難しいことがあります。

## メルカリ向きか、買取業者向きか

| 向いている方法 | 品物の特徴 |
| --- | --- |
| メルカリ向き | 小さく発送しやすい、相場を調べやすい、写真で状態を伝えやすい |
| 買取業者向き | 数が多い、専門知識が必要、大型、壊れやすい、高額品 |
| 処分検討 | 破損が大きい、需要が少ない、衛生面の問題がある |

## よくある質問

### 古い物でも売れることはありますか？

あります。古い物でも、専門性、希少性、ブランド、保存状態、コレクター需要によって査定対象になることがあります。ただし、必ず売れるとは限りません。

### 汚れている物は捨てた方がよいですか？

軽い汚れなら査定対象になることがありますが、破損やカビ、強いにおいがある場合は難しいことがあります。無理にきれいにしようとして傷めないよう注意してください。

### 家族に価値がわからない物はどうすればよいですか？

写真を撮り、メーカー名、型番、素材、付属品を確認します。判断できない場合は、専門店や買取業者の査定対象か調べてから処分を考えましょう。

## まとめ

実家の片付けで出てくる物は、見た目だけでは価値を判断しにくいことがあります。

捨てる前に、売れる可能性があるものを分け、状態や付属品を確認しておくと、後悔しにくい整理につながります。
