# 固定ページ 仮ピラー投入メモ（廃止）

## 現在の扱い

このフォルダの固定ページ案は、初期検討用の古い案です。

現在は以下の方針に変更しています。

- ピラーは固定ページではなく投稿記事で作る
- グロナビは代表ピラー投稿へリンクする
- 売れるものの整理ピラーは新規 `/sellable-items/` を作らず、既存記事 `/sellable-disused-items/` をリライトして使う
- カテゴリページは記事棚として使い、当面 `noindex, follow`

このフォルダ内の原稿は、そのままWordPressへ投入しません。必要な要素だけ、`drafts/pillars/` のリライト設計に移して使います。

## WordPress設定

- 投稿ではなく固定ページで作成する（廃止）
- URLは英語スラッグにする
- グロナビは固定ページへリンクする
- カテゴリページにはリンクしない
- 仮ピラー公開後、各ページから関連記事へリンクする

## 画像

画像は最初から完璧に作り込まなくてよいです。

まずは各固定ページに1枚ずつ、1200 x 630の横長アイキャッチを入れます。画像方針は `strategy/image-policy.md` を参照します。
