# 着物・帯は売れる？実家の呉服を捨てる前に確認したいこと

実家の片付けを進める中で、タンスに眠っていた大量の着物や帯が見つかることがあります。

「着物の知識が全くないから、価値があるのかどうかわからない」
「シミもあるし、捨てるしかないのかな…でも親の思い出だから忍びない」

着物に馴染みがない私たちにとって、実家の古い着物や帯を前に戸惑うのはごく自然なことです。

パッと見ただけでは、それが「価値ある伝統工芸品」なのか「大量生産された安価なもの」なのか、素人にはまず見分けられません。

しかし、捨てる判断を下す前に「売れる着物の基準」を少し知っておくだけで、価値あるものをうっかり捨てて後悔するのを防ぐことができます。

この記事では、親の着物や帯を見つけたけれど価値判断ができない人に向けて、実家の着物が売れるのかどうか、処分する前に確認すべきポイントを解説します。

着物は無理に自分で価値を判断せず、『プロに見てもらう』のが一番の近道です。まずはクリーニングなどもせず、そのままの状態を確認してみましょう。

**この記事でわかること**
- 売れる着物と売れない着物の見分け方
- 証紙や素材、状態など査定でチェックされるポイント
- 帯や和装小物（草履・バッグなど）の扱い方
- 古い着物を適正に手放すための具体的な手順

## 実家の着物・帯は売れる？捨てる前に知っておきたい結論

結論から言うと、**実家の古い着物や帯は「売れる（高い価値がつく）可能性」が十分にあります。**決して自己判断で捨ててはいけません。

- **素人の自己判断は禁物**：一見古びて見えても、数万円〜数十万円の価値が隠れていることがあります。
- **まずはプロの無料査定へ**：価値を見分けるには専門知識が必要です。そのままの状態でプロに見てもらいましょう。

「価値を確認し、値段がつかなかったものだけを処分する」というのが、最も損をしない進め方です。

### 素人判断で捨ててしまうのはもったいない

着物の価値は、見た目の新しさや柄の好みだけで決まるものではありません。**プロの目から見ると、一見古びて見える着物でも数万円から数十万円の価値が隠れている**ことがあります。

特に、有名作家の作品や伝統工芸品、希少な産地で織られた着物は、中古市場でも高い需要があります。また、着物だけでなく、帯や和装小物（帯留め、かんざしなど）にも予想外の値段がつくケースは珍しくありません。素人判断で「古いから」「シミがあるから」とゴミ袋に入れてしまう前に、一度立ち止まることが重要です。

### 「売れるか分からない」なら、まずはプロの無料査定へ

着物の本当の価値を見極めるには、織り方や染め方、産地を特定する専門的な知識が不可欠です。もしあなたが着物の知識を持っていないのであれば、「売れるのか、売れないのか」を自分で判断する必要はありません。

**最も確実で安全な方法は、着物専門の買取業者による「無料査定」を利用すること**です。

多くの買取業者は、査定料や出張料、キャンセル料を無料に設定しています。まずはプロの目で価値を判断してもらい、値段がつけば買い取ってもらい、買取不可であればその後に処分方法を考える、という手順を踏むのが、最も損をしない実家の着物整理の進め方です。

## どんな着物・帯なら売れる？買取されやすい特徴と見分け方

「すべてプロにお任せする」とはいえ、査定員がどのようなポイントを見ているのかを知っておくと安心です。実家の着物の中で、特に値段がつきやすいものの特徴をまずは一覧で確認しましょう。

| チェックポイント | 高値がつきやすい特徴（売れる） | 買取が難しい特徴（値段がつきにくい） |
| :--- | :--- | :--- |
| **① 素材** | 正絹（しょうけん・絹100%） | ウール、化繊（ポリエステル） |
| **② 証紙・証明書** | あり（伝統工芸品や有名作家の証明） | なし |
| **③ 帯・和装小物** | 有名産地の帯、精巧な帯留め・かんざしなど | - |
| **④ 状態** | 良好（※価値ある品ならシミありでも可） | 生地が弱って破れそうな状態、ひどいカビ |

それぞれの詳しい見分け方について解説します。

### 素材が「正絹（しょうけん）」であるか

着物の査定において、最も重要な要素の一つが「素材」です。**絹100%で作られた「正絹（しょうけん）」の着物は、中古市場でも高く評価されます。**

正絹は特有の光沢と滑らかな肌触りがあり、着心地が良いため着物愛好家からの需要が絶えません。ノーブランドであっても、正絹であれば数千円程度の値段がつく可能性があります。見分けるのが難しい場合は、触った時にしっとりとした柔らかさがあるかどうかが一つの目安になりますが、最終判断はプロに任せましょう。

### 産地や作家を証明する「証紙」がついているか

購入時に着物と一緒についてくる「証紙（しょうし）」は、着物の価値を証明する重要なアイテムです。**大島紬や結城紬などの伝統工芸品、または有名作家の作品であることを証明する証紙があれば、買取価格は大幅に跳ね上がります。**

証紙は、小さな布切れや紙片のような形でたとう紙（着物を包む紙）の中に入っていることが多いです。「ただの切れ端」だと思って捨ててしまわないよう、タンスの引き出しやたとう紙の中は隅々まで確認してください。

### 着物だけでなく「帯」や「和装小物」も売れる

着物本体に目が行きがちですが、**一緒に保管されている「帯」や「和装小物」もしっかりと買取の対象になります。**

特に、袋帯や名古屋帯などで西陣織などのブランド帯であれば、着物本体よりも高値がつくケースも少なくありません。また、帯留め、帯締め、草履、和装バッグ、さらには反物（仕立てる前の布地）もまとめて買い取ってもらえます。着物一式が揃っている場合は、セットで査定に出すことで評価額がアップすることもあります。

### シミやカビなど「状態」はどこまで許容される？

長年タンスにしまっていた着物には、シミや黄ばみ、カビが発生していることがよくあります。「こんなに汚れていたら売れないだろう」と思いがちですが、**有名産地の着物や骨董的価値のあるアンティーク着物であれば、シミがあっても買い取ってもらえる**可能性があります。

シミ抜きや修繕の技術を持つ買取業者であれば、自社でメンテナンスして再販できるためです。ただし、汚れの程度によっては値段がつかない（無料引き取りになる）こともあります。まずはご自身で捨てずに、プロの判断を仰ぎましょう。

## 買取不可になりやすい実家の着物・帯の条件

査定に出せばどんな着物でも必ず売れるわけではありません。業者側の再販が難しいため、買取不可になりやすい着物の条件もあらかじめ知っておきましょう。

### ウールや化繊など大量生産された着物

正絹が高く評価される一方で、**ポリエステルなどの化学繊維（化繊）やウール素材の着物は、買取を断られるケースが多い**です。

これらの素材は普段着として大量生産されており、元々の販売価格が安価です。そのため中古市場での需要も低く、値段がつきにくくなっています。「昔、練習用や普段着としてたくさん買った」という着物は、この素材に当てはまる可能性が高いでしょう。

### 喪服やサイズの小さすぎる着物

黒紋付などの「喪服」は、かつては嫁入り道具として誂えるのが一般的でしたが、現在ではレンタルで済ませる人がほとんどです。**需要が極端に低く、家紋が入っていることも相まって、たとえ正絹であっても買取不可になるのが一般的**です。

また、昔の日本人の体型に合わせて作られた「サイズの小さすぎる着物（身丈が短いなど）」も、現代人が着こなすのが難しいため評価が下がります。

### 著しい劣化・強い匂いがあるもの

多少のシミであれば買い取ってもらえる可能性があるとお伝えしましたが、**修復不可能なほどの「ひどいカビ」や「広範囲の虫食い」、そして「強い防虫剤（ナフタリンなど）の匂い」が染み付いているものは買取不可になりやすい**です。

保管環境が悪く、生地そのものが弱ってしまっている（触ると破れそうなど）状態では、リメイクの素材としても使えないため、引き取りも断られることがあります。その場合は、自治体のルールに従って一般ごみ（布ごみなど）として処分するか、不用品回収業者に依頼して手放すことになります。

### 無理にクリーニングや修繕をしない

着物にシミや汚れを見つけると「綺麗にしてから査定に出した方が高く売れるのでは」と考えがちですが、**事前のクリーニングや修繕は絶対に避けてください。**

着物のクリーニング（丸洗いやすすぎ）には数千円から数万円の高い費用がかかります。せっかく綺麗にしても、クリーニング代以上に査定額がアップすることはほとんどなく、結果的に赤字（損）になってしまいます。優良な買取業者は自社で安価にメンテナンスするルートを持っているため、現状のまま査定に出すのが鉄則です。

### たとう紙に入れたまま、広げずに保管する

タンスの中を整理する際、どんな着物が入っているか確認しようと無理に広げてしまうのは危険です。**慣れない人が着物を広げると、正しく畳み直すことができず、新たなシワや型崩れの原因になります。**

シワがつくと査定時の印象が悪くなり、評価が下がる可能性があります。たとう紙（和紙の包み紙）に入っているものは、そのまま無理に広げず、査定員が来るまで大切に保管しておきましょう。

### 証紙や小物は必ずセットにしておく

前述の通り、**着物の価値を証明する「証紙」や、一緒に使っていた「和装小物」は必ずセットにして査定に出す**ようにしてください。

「証紙がないと買い取れない」というわけではありませんが、あるかないかで数万円単位の差が出ることがあります。また、小物類もバラバラに査定に出すより、着物や帯と合わせて「一式」として見てもらうことで、トータルでの買取額アップが期待できます。タンスの中や桐箱の隅に忘れ物が落ちていないか、最後にもう一度確認しておきましょう。

## 実家の着物買取で失敗しない業者の選び方

いざ査定に出すとなっても、「どの業者に頼めばいいかわからない」「悪徳業者に騙されないか不安」という方は多いでしょう。最後に、安心して任せられる業者の選び方を解説します。

### 「着物専門」の査定員がいる業者を選ぶ

リサイクルショップや総合買取店でも着物を扱っていることがありますが、**着物は専門知識がないと適正な価値を見極めることができないため、「着物買取を専門にしている業者」を選ぶことが鉄則**です。

総合買取店の場合、単に「古い布切れ」として重量で買い取られてしまい、本来なら数万円の価値がある着物が数百円にしかならないこともあります。着物専門の査定員が在籍し、豊富な買取実績を公開している業者を選びましょう。

### 持ち運びが不要な「出張買取」がおすすめ
