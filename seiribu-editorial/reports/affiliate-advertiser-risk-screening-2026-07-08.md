# セイリ部 専門買取広告主リスク一次判定

作成日: 2026-07-08  
対象: `affiliate-specialty-buyback-candidates-2026-07-08.md` に整理した専門買取案件  
目的: 評判リスク、掲載リスク、提携価値の薄い広告主を見極めるための一次スクリーニング

## 前提

「評判が悪い」と断定するには、Notion本文だけでは足りません。  
本資料では、まずNotion内の案件本文、成果条件、単価、リンク状態、セイリ部との文脈一致から、**掲載前に慎重に扱うべき広告主**を抽出します。

外部評判は変動するため、本番掲載前に以下を確認します。

- 公式サイトの運営会社情報、古物商許可、問い合わせ先
- Googleビジネスプロフィール、レビューサイト、SNSでの直近の不満傾向
- 「査定額が低い」「返送料が高い」「電話がしつこい」「キャンセルしづらい」などの反復クレーム
- ASP管理画面上の成果条件、否認条件、禁止表現
- LPのリンク切れ、運営会社変更、サービス終了

## 判定基準

| 判定軸 | 見るポイント |
|---|---|
| 読者適合度 | 実家整理、遺品整理、生前整理、価値不明品の処分に合うか |
| 信頼材料 | 運営会社、実店舗、上場企業、実績、問い合わせ先が確認しやすいか |
| 成果条件 | 申込だけか、査定完了か、買取成立か。最低買取金額条件が厳しくないか |
| 否認リスク | 地域限定、重複、対象外品、LINE申込除外などで成果が落ちやすくないか |
| 表現リスク | 「なんでも買取」「必ず高価買取」などが使えない案件か |
| 単価と工数 | 記事化・提携・管理に見合う単価か |
| 評判確認必要度 | 外部口コミを見ないと読者に薦めづらいか |

## 使わない方がよい訴求

広告主の評判以前に、セイリ部の読者保護として避けたい表現です。

- 「高く売れる業者ランキング」と断定する
- 「どんな状態でも売れる」と書く
- 「捨てるより必ず得」と書く
- 口コミ確認なしに「評判が良い」と書く
- 実体験していないのに「使ってみた」と書く
- 成果対象外の電話、LINE、店舗持ち込みへ誘導する
- 対象外エリアを全国対応のように書く

## 次の調査手順

1. まず `保留・低優先` の案件は記事候補から外す。
2. `掲載できるが慎重` の案件は、記事で使う前に外部評判と公式条件を確認する。
3. `現時点で掲載しやすい` 案件だけで、カメラ、骨董、人形、ホビー、レコード、楽器の記事構成を作る。
4. 外部評判調査では、広告主名単体ではなく `広告主名 + 買取 + 評判 + 悪い + 査定額 + キャンセル` のように具体語で確認する。
5. 口コミはアフィリエイト記事だけで判断せず、レビューサイト、Google口コミ、SNS、公式の運営会社情報を分けて見る。

## 一次結論

現時点で、明確に後回しでよいのは **モデモール、古書大黒堂、ブックオフ公式宅配買取、ミニカー買取王国、カーナベル、アメトイ買取王国、フィッシングコレクト** です。  
これらは「悪評が確認された」というより、リンク状態、単価、成果条件、読者適合度の面で、セイリ部の初期収益記事に優先する理由が弱い案件です。

逆に、**バイセル、GoodDeal、王の洞窟、ウリエル系、YTHカメラ** は使える可能性がありますが、掲載前の確認コストが高い案件として扱います。  
**COYASH** は外部口コミで人形・不用品訴求とのズレが目立ったため、現時点では初期主力から外します。
