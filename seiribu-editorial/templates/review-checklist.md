# レビュー用チェックリスト

## 制作フロー

- [ ] 標準フロー対象か軽量フロー対象か判断した
- [ ] 構成承認前に本文を書いていない
- [ ] 導入文をH2本文とは別に作成した
- [ ] 標準フローではH2ごとに本文を作成した
- [ ] 各フェーズで人間確認を挟んだ

## 検索意図

- [ ] タイトルと本文が一致している
- [ ] 読者の検索意図に答えている
- [ ] 結論が前半でわかる

## 表現

- [ ] 「必ず」「絶対」などの断定がない
- [ ] 根拠のない相場金額がない
- [ ] 実体験ではない体験談がない
- [ ] 業者っぽい煽りがない

## 内容

- [ ] 判断表がある
- [ ] 捨てる前チェックリストがある
- [ ] 買取不可だった場合の選択肢がある
- [ ] FAQがある

## 内部リンク

- [ ] 関連記事へのリンクが自然に入っている
- [ ] ハブ記事への導線がある
- [ ] 無理なリンク詰め込みがない

## 公開前

- [ ] frontmatterがある
- [ ] `human_review_required: true` がある
- [ ] 広告表記が必要な場合は明記されている
- [ ] 人間が最終確認した
