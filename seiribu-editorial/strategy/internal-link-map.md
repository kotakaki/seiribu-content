# 内部リンク方針

## ハブ記事候補

| 役割 | 記事タイトル案 | リンクさせる場面 |
| --- | --- | --- |
| 心理入口 | 物を捨てられない人の心理と家族ができる整理の進め方 | 心理・家族の悩み系記事のまとめ |
| 親との話し合い | 親が物を捨てさせてくれないときの話し方 | 実家の荷物を勝手に捨てられない場面 |
| 総合ハブ | 実家の片付けで売れるもの一覧 | 品目別記事の冒頭またはまとめ |
| 失敗防止 | 買取基準に満たないと言われた理由 | 売れにくい条件を説明した後 |
| 行動判断 | 実家の不用品は買取と回収どちらが先？ | 処分方法に迷う読者向け |
| 処分導線 | 買取不可だったものを処分する方法 | 買取不可の説明後 |

## 心理系記事からの導線

心理系ロングテールは、セイリ部の初期流入を作る入口として重視します。

ただし、心理記事だけで完結させず、以下の順で内部リンクを設計します。

1. 物を捨てられない理由を理解する
2. 勝手に捨てない、家族で確認する
3. 売る・残す・処分に分ける
4. 価値がありそうな物は捨てる前に確認する
5. 買取基準、売れるもの、処分方法へ進む

## ルール

- 内部リンクは読者の次の判断に役立つ場所に入れる
- 1記事あたりの内部リンクは自然な範囲にする
- 同じアンカーテキストを不自然に繰り返さない
- 心理記事では、精神論だけで終わらせず実務記事へ送る
