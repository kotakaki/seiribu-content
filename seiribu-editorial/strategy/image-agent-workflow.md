# セイリ部 画像生成・Canva連携ワークフロー

## 結論

画像生成用のエージェントは作った方がよいです。

現在のセイリ部Codexは、記事本文を書くエージェントではなく、完成した記事に画像を付けるためのエージェントとして運用します。

画像は記事本文とは別工程にして、記事ごとに以下を作ります。

- アイキャッチ画像の目的
- 読者に伝えたい状況
- 避けたい表現
- 画像概要
- 画像制作エンジン用の最終プロンプト
- ファイル名
- WordPress画像タイトル
- ALTテキスト
- キャプション候補
- 設置位置
- ロゴ配置位置
- Canva用のタイトル文言
- Canvaテンプレへの流し込み指示

## 役割分担

| 役割 | 担当すること | 担当しないこと |
| --- | --- | --- |
| 記事エージェント | 記事内に画像概要、設置位置、ALT候補、避けたい表現を書く | 画像制作手法の最終判断、完成プロンプトの固定 |
| セイリ部Codex / 画像制作エンジン | 進捗管理、画像タイプ判定、最終プロンプト化、構図、トーン、NG表現、画像タイトル、ALT | 記事本文の新規執筆、全文リライト |
| Canva整形エージェント | タイトル文字入れ、ロゴ配置、テンプレ統一 | 画像の意味づけ |
| 人間 | 最終確認、違和感チェック、公開判断 | なし |

## 基本方針：記事内の指定は「制作ブリーフ」として扱う

記事内に書く画像指定は、完成プロンプトではなく制作ブリーフです。

記事エージェントは、本文理解に必要な画像の目的と概要までを書きます。画像制作エンジンは、その概要をもとに、アイキャッチ、情景イラスト、写真風素材、記事内図解のどれで作るべきかを判断し、必要に応じてプロンプトを書き換えます。

特に日本語ラベル入りの図解は、画像生成AIへ丸投げしません。文字崩れや誤字を避けるため、Canva、Pillow、SVGなどのレイアウト制御ができる方法で作ります。

## Canva連携でできること

Canva連携は可能です。

主に以下の用途に向いています。

- 既存のCanvaテンプレを複製する
- タイトルやサブタイトルを差し替える
- ページやデザイン内容を確認する
- 画像サイズを変える
- 複数記事のアイキャッチを同じデザインルールで揃える

## おすすめ運用

### 1. Canvaで共通テンプレを作る

まず人間がCanvaで、セイリ部用のアイキャッチテンプレを1つ作ります。

必要な要素:

- セイリ部ロゴ
- 記事タイトル枠
- 短い補足文枠
- 画像エリア
- 色とフォントの固定

### 2. 記事ごとに画像指示を作る

完成記事を確認したあと、本文とは別に画像指示を作ります。

```text
記事タイトル:

画像の目的:

読者に伝えたい状況:

入れたい要素:

避けたい要素:

画像概要（記事エージェントが書く制作ブリーフ）:

画像タイプ候補:

画像制作エンジン用の最終プロンプト:

ファイル名:

WordPress画像タイトル:

ALTテキスト:

キャプション候補:

設置位置:

ロゴ配置位置:

Canvaタイトル:

Canvaサブ:
```

### 3. 画像素材を生成する

画像生成では、背景や状況が伝わる素材を作ります。

セイリ部では以下のトーンを守ります。

- 日本の実家らしさ
- 親子・家族の確認作業
- 古い物、趣味品、コレクション
- 明るく清潔
- 不安を煽らない
- 買取業者っぽくしすぎない

### 3-1. 画像制作エンジンの判断フロー

記事内の画像概要を受け取ったら、以下の順番で判断します。

1. 画像の役割を判定する
   - アイキャッチ
   - 記事内イメージ
   - 記事内図解
   - 写真風素材
2. 画像制作方法を選ぶ
   - 情景イラストや写真風素材は画像生成エンジンで作る
   - 日本語ラベルや比較表を含む図解は、Canva、Pillow、SVGなどでレイアウト生成する
   - アイキャッチは、必要に応じて画像生成素材とCanvaテンプレを組み合わせる
3. 記事内の概要を最終プロンプトに変換する
   - 読者層、場所、人物、品物、表情、光、避けたい表現を補う
   - 画像内文字の有無を明確にする
   - ロゴ配置の余白を確保する
4. 完成後に確認する
   - 文字崩れがない
   - ロゴが見出し、人物の顔、重要な品物を邪魔していない
   - 暗い遺品整理、ゴミ屋敷、高額買取広告のように見えない
   - ファイル名、ALT、キャプション、設置位置をREADMEに残す

## 画像タイプ別ルール

| 画像タイプ | 主な用途 | 制作方法 | 文字 | サイズ |
| --- | --- | --- | --- | --- |
| アイキャッチ | 記事全体の第一印象を作る | 温かい編集イラスト調を基本に、必要ならCanvaでタイトルを載せる | 素材画像には原則入れない。タイトルはCanva側で管理する | 1200 x 630 |
| 記事内イメージ | H2直下で読者の状況理解を助ける | 情景イラストを基本にする | 原則入れない | 1200 x 800 |
| 写真風素材 | 実物感や生活感を強く見せたい場合だけ使う | 写真風生成または写真素材 | 入れない | 1200 x 630 または 1200 x 800 |
| 記事内図解 | 手順、比較、分類、判断基準を説明する | Canva、Pillow、SVGなどでレイアウト生成する | 必要な日本語を正確に入れる | 1200 x 800 |

### アイキャッチの判断基準

アイキャッチは、検索ユーザーが「これは自分の状況に近い」と感じることを優先します。

基本は、実家の室内で親子や家族が古い物、段ボール、写真、本、カメラ、家具などを落ち着いて確認している編集イラスト調にします。

写真風にするのは、実物感を出す必要が強い場合だけです。写真風は暗さ、汚さ、業者広告感が出やすいため、清潔で明るい生活感に寄せます。

アイキャッチのブランド型は、左上にセイリ部ロゴ、上部中央に大きな見出し、その直下にサブタイトル、下部に人物や状況イラストを置く構成を基本にします。大きな白カードで見出しを囲うのではなく、淡い背景と余白で読ませます。

フォントは、見出しにUDモトヤアポロ太字、サブタイトルにNoto Sans JP Regularを使います。制作環境にUDモトヤアポロがない場合は、Noto Sans JP Boldなどで暫定制作し、最終Canvaまたは本番制作環境で差し替えます。

### 記事内図解の判断基準

次の条件に当てはまる場合は、画像生成AIではなくレイアウト生成で作ります。

- 日本語の見出し、表、ラベルを正確に入れる必要がある
- 手順や比較を読ませることが目的である
- 3分類、4ステップ、費用比較など、情報構造が主役である
- 文字崩れがSEO記事の信頼性を下げる可能性がある

### 記事エージェントが本文内に書く指定の最小形

```text
> [!CMS担当者へ：ここに画像を挿入]
> 種類：アイキャッチ / 記事内イメージ / 写真風素材 / 記事内図解
> 目的：この画像で読者に伝えたいこと
> ALT：画像を見られない人にも伝わる説明
> 指示：入れたい要素、避けたい要素、必要なラベル
```

この指定だけで足りない場合は、画像制作エンジン側で補います。記事エージェントは、最終プロンプトを過度に固定しません。

### 4. Canvaで整える

生成画像をそのまま使うより、Canvaで統一感を出します。

- 画像の上にタイトルを載せる
- セイリ部ロゴを入れる
- 文字サイズを統一する
- 余白を揃える
- 記事カテゴリごとに色を少し変える

| ページ | 画像素材 | Canvaタイトル |
| --- | --- | --- |
| 実家の片付け | 親の家で荷物を3分類している親子 | 実家の片付けは何から始める？ |
| 売れるものの整理 | 古い本、カメラ、レコード、ブランド品が並ぶ段ボール | 実家で売れるもの一覧 |
| 買取・処分の進め方 | 買取、処分、確認中の箱に分けた荷物 | 買取不可だったものはどうする？ |
| 家族の悩み | 親子が古い物を見ながら話し合う | 親が物を捨てさせてくれないとき |

## 画像生成エージェントのルール

### 必ず守ること

- 実家整理の読者に近い生活感を出す
- 40代から60代の子供世代を想定する
- 親の物を尊重している雰囲気にする
- 「捨てる」より「確認する」「分ける」を見せる
- 可能な範囲で `assets/images/brand/seiribu-logo.png` のロゴを、本文や図解ラベルを邪魔しない余白に入れる
- 画像内の文字は原則入れない

### 避けること

- ゴミ屋敷のように汚く見せる
- 高額査定や札束を強調する
- 怖い遺品整理の雰囲気にする
- ミニマリスト風の真っ白な部屋にする
- 若者向けのポップすぎるデザインにする
- ロゴを大きく置きすぎる、見出し・人物の顔・重要な品物に重ねる

## 画像タイトル・ALT生成ルール

画像を作るときは、必ず以下も同時に作ります。

| 項目 | ルール |
| --- | --- |
| ファイル名 | 英数字、ハイフン区切り。記事スラッグと用途を入れる |
| WordPress画像タイトル | 画像の内容がわかる日本語。記事タイトルの丸写しにしない |
| ALTテキスト | 画像を見られない人にも内容が伝わる説明。80字前後まで |
| キャプション | 必要な場合だけ。本文の理解を助ける短い説明 |
| 設置位置 | アイキャッチ、見出し直下、表の前後などを明記 |
| ロゴ配置位置 | 右下・左上など、本文や図解ラベルを邪魔しない余白を明記 |

### ALTで避けること

- キーワードを不自然に詰め込む
- 「画像」「イラスト」だけで終わる
- 記事タイトルをそのまま繰り返す
- 画像にない情報を書く
- 「高く売れる」など根拠のない表現を入れる

### ALTの良い例

実家のリビングで親子が段ボールの荷物を確認し、古い本や写真を整理している様子

### ALTの悪い例

実家の整理 売れるもの 買取 処分 おすすめ

## Canvaテンプレの推奨サイズ

- アイキャッチ: 1200 x 630
- 正方形SNS用: 1200 x 1200
- 記事内図解: 1200 x 800

## 運用開始に必要なもの

1. Canvaのセイリ部用テンプレURL
2. ロゴ画像
3. 使用フォントと色
4. 画像サイズ
5. カテゴリごとの色ルール

## 最初のおすすめ

まずは、仮ピラー4本だけCanvaテンプレで揃えます。

その後、主力記事リライト時にアイキャッチを更新します。

最初から全記事の画像を作り直すと工数が大きくなるため、以下の順番がよいです。

1. 仮ピラー4本
2. 実家の片付けで売れるもの一覧
3. 買取基準に満たないとは？
4. 信頼できる買取業者の見分け方
5. 悪質な訪問買取業者の特徴
