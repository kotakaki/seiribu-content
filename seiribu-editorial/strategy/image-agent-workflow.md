# セイリ部 画像生成・Canva連携ワークフロー

## 結論

画像生成用のエージェントは作った方がよいです。

現在のセイリ部Codexは、記事本文を書くエージェントではなく、完成した記事に画像を付けるためのエージェントとして運用します。

画像は記事本文とは別工程にして、記事ごとに以下を作ります。

- アイキャッチ画像の目的
- 読者に伝えたい状況
- 避けたい表現
- 画像生成プロンプト
- ファイル名
- WordPress画像タイトル
- ALTテキスト
- キャプション候補
- 設置位置
- Canva用のタイトル文言
- Canvaテンプレへの流し込み指示

## 役割分担

| 役割 | 担当すること | 担当しないこと |
| --- | --- | --- |
| セイリ部Codex | 進捗管理、画像プロンプト、構図、トーン、NG表現、画像タイトル、ALT | 記事本文の新規執筆、全文リライト |
| Canva整形エージェント | タイトル文字入れ、ロゴ配置、テンプレ統一 | 画像の意味づけ |
| 人間 | 最終確認、違和感チェック、公開判断 | なし |

## Canva連携でできること

Canva連携は可能です。

主に以下の用途に向いています。

- 既存のCanvaテンプレを複製する
- タイトルやサブタイトルを差し替える
- ページやデザイン内容を確認する
- 画像サイズを変える
- 複数記事のアイキャッチを同じデザインルールで揃える

## おすすめ運用

### 1. Canvaで共通テンプレを作る

まず人間がCanvaで、セイリ部用のアイキャッチテンプレを1つ作ります。

必要な要素:

- セイリ部ロゴ
- 記事タイトル枠
- 短い補足文枠
- 画像エリア
- 色とフォントの固定

### 2. 記事ごとに画像指示を作る

完成記事を確認したあと、本文とは別に画像指示を作ります。

```text
記事タイトル:

画像の目的:

読者に伝えたい状況:

入れたい要素:

避けたい要素:

生成プロンプト:

ファイル名:

WordPress画像タイトル:

ALTテキスト:

キャプション候補:

設置位置:

Canvaタイトル:

Canvaサブ:
```

### 3. 画像素材を生成する

画像生成では、背景や状況が伝わる素材を作ります。

セイリ部では以下のトーンを守ります。

- 日本の実家らしさ
- 親子・家族の確認作業
- 古い物、趣味品、コレクション
- 明るく清潔
- 不安を煽らない
- 買取業者っぽくしすぎない

### 4. Canvaで整える

生成画像をそのまま使うより、Canvaで統一感を出します。

- 画像の上にタイトルを載せる
- セイリ部ロゴを入れる
- 文字サイズを統一する
- 余白を揃える
- 記事カテゴリごとに色を少し変える

| ページ | 画像素材 | Canvaタイトル |
| --- | --- | --- |
| 実家の片付け | 親の家で荷物を3分類している親子 | 実家の片付けは何から始める？ |

## 画像生成エージェントのルール

### 必ず守ること

- 実家整理の読者に近い生活感を出す
- 40代から60代の子供世代を想定する
- 親の物を尊重している雰囲気にする
- 「捨てる」より「確認する」「分ける」を見せる
- 画像内の文字は原則入れない

### 避けること

- ゴミ屋敷のように汚く見せる
- 高額査定や札束を強調する
- 怖い遺品整理の雰囲気にする
- ミニマリスト風の真っ白な部屋にする
- 若者向けのポップすぎるデザインにする

## 画像タイトル・ALT生成ルール

画像を作るときは、必ず以下も同時に作ります。

| 項目 | ルール |
| --- | --- |
| ファイル名 | 英数字、ハイフン区切り。記事スラッグと用途を入れる |
| WordPress画像タイトル | 画像の内容がわかる日本語。記事タイトルの丸写しにしない |
| ALTテキスト | 画像を見られない人にも内容が伝わる説明。80字前後まで |
| キャプション | 必要な場合だけ。本文の理解を助ける短い説明 |
| 設置位置 | アイキャッチ、見出し直下、表の前後などを明記 |

### ALTで避けること

- キーワードを不自然に詰め込む
- 「画像」「イラスト」だけで終わる
- 記事タイトルをそのまま繰り返す
- 画像にない情報を書く
- 「高く売れる」など根拠のない表現を入れる

### ALTの良い例

実家のリビングで親子が段ボールの荷物を確認し、古い本や写真を整理している様子

### ALTの悪い例

実家の整理 売れるもの 買取 処分 おすすめ

## Canvaテンプレの推奨サイズ

- アイキャッチ: 1200 x 630
- 正方形SNS用: 1200 x 1200
- 記事内図解: 1200 x 800

## 運用開始に必要なもの

1. Canvaのセイリ部用テンプレURL
2. ロゴ画像
3. 使用フォントと色
4. 画像サイズ
5. カテゴリごとの色ルール

## 最初のおすすめ

まずは、仮ピラー4本だけCanvaテンプレで揃えます。

その後、主力記事リライト時にアイキャッチを更新します。

最初から全記事の画像を作り直すと工数が大きくなるため、以下の順番がよいです。

1. 仮ピラー4本
2. 実家の片付けで売れるもの一覧
3. 買取基準に満たないとは？
4. 信頼できる買取業者の見分け方
5. 悪質な訪問買取業者の特徴
