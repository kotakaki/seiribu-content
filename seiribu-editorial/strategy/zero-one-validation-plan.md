# ゼロイチ検証方針

## 目的

まずは完成度の高い大規模サイトを作るより、最低限の棚と記事を入れて、Googleがどの検索意図に反応するかを確認します。

最初の目標は、順位ではなく以下です。

- Search Consoleで表示回数が出る
- クリックされる記事が出る
- どのクエリに反応があるか見える
- 反応がある棚を次に厚くできる

## 現時点の観察

記事削減、放置、未インデックス記事がある状態でも、過去90日では心理・家族の悩み系記事に流入が残っています。

特に入口として反応がある記事:

| 記事 | 観察 |
| --- | --- |
| 急に物を減らしたくなる心理とは？ | 最も流入が残っている |
| 実家の物が多すぎてイライラした時の対処法 | 実家整理と心理の接点として強い |
| 物を捨てられない男性の心理とは？ | 過去に上位表示実績がある |
| 物を減らすと楽になる！ | 滞在時間が長く、悩み入口として残す価値がある |
| 断捨離をする人が「頭おかしい」と思われがちな理由 | 心理系ロングテールとして反応あり |

このため、短期の勝ち筋は「心理・家族の悩み系ロングテールで入口を作り、実家整理と買取判断へ流す」ことです。

## やらないこと

- 商標記事や口コミ記事を最初から量産しない
- 実体験がないサービスレビューを書かない
- 心理記事をミニマリズムや精神論だけで終わらせない
- カテゴリトップをピラー代わりにindexさせない
- 薄い仮ピラーをindexさせない

## 最小の棚

最初に用意する棚は以下です。

| 棚 | 役割 | index方針 |
| --- | --- | --- |
| トップページ | セイリ部の説明と主要導線 | index |
| 家族の悩みピラー | 心理系流入を受ける代表記事 | 完成後index |
| 実家の片付けピラー | 実家片付けの全体手順 | 完成後index |
| 売れるものの整理ピラー | 既存記事 `/sellable-disused-items/` をピラー化し、捨てる前の価値確認へつなぐ | index |
| 買取・処分の進め方ピラー | 売れない時の判断 | 完成後index |
| カテゴリページ | 読者用の記事棚 | noindex, follow |
| タグページ | 補助分類 | noindex, follow |

ピラーは投稿記事で作ってよいです。完成前の仮ピラーはnoindexで公開し、読者導線と内部リンクの受け皿として使います。index可能な水準に達したらindexへ切り替えます。

グロナビはピラー投稿へリンクし、カテゴリページへは直接リンクしません。カテゴリ名とグロナビ名は似たテーマでも完全一致させず、カテゴリスラッグは英語で統一します。

## 最初に強化する記事群

1. 急に物を減らしたくなる心理とは？
2. 実家の物が多すぎてイライラした時の対処法
3. 物を捨てられない男性の心理とは？
4. 物を減らすと楽になる！
5. 断捨離をする人が「頭おかしい」と思われがちな理由
6. 親が物を捨てさせてくれないときの話し方
7. 実家の片付けで売れるもの一覧
8. 「買取基準に満たない」とは？

心理記事をリライトする時は、最後に必ず以下へ接続します。

- 勝手に捨てない
- 本人や家族に確認する
- 売る・残す・処分に分ける
- 価値がありそうな物は捨てる前に確認する
- 実家整理ピラー、売れるものピラー、買取判断記事へ送る

## 20記事前後のゼロイチセット

最初は20記事前後で検証します。

| 種類 | 本数 | 内容 |
| --- | ---: | --- |
| ピラー | 4本 | 家族と心の整理、実家の整理、売れるもの、買取・処分 |
| 心理・家族の悩み | 6から8本 | 既存流入記事と復元記事 |
| 実家整理 | 3から4本 | 実家じまい、兄弟分担、一人っ子、親の家 |
| 売れるもの・買取判断 | 4から5本 | 売れるもの、本、買取基準、不用品と不要品 |
| 安全確認 | 2から3本 | 信頼できる業者、悪質訪問買取、遺品整理の不安 |

## 計測

公開後は、Search Consoleで以下を見ます。

- 表示回数が出たクエリ
- クリックされたページ
- インデックスされたページ
- 404に流入していないか
- 心理記事から実家整理・買取判断記事へ回遊しているか

反応が出た棚を優先して、30から40記事の専門サイト骨格へ広げます。
