# セイリ部 画像方針

## 基本方針

画像は、最初から作り込みすぎなくてよいです。

仮ピラーでは、読者が「これは自分の状況に近い」と感じることを優先します。おしゃれな収納写真より、実家の荷物、段ボール、古い物、親子の確認作業が伝わる画像が向いています。

## 画像で伝えること

- 実家の荷物を整理している
- 捨てる前に価値を確認している
- 親子・家族で話し合っている
- 売る・残す・処分するに分けている
- 不安を煽らず、落ち着いて進められる

## 避けたい画像

- 真っ白で生活感がないミニマリスト部屋
- 高級感だけが強い買取業者風の画像
- 汚部屋、ゴミ屋敷感が強すぎる画像
- 暗い遺品整理、孤独、恐怖を煽る画像
- 現金や高額査定を強調する画像

## 仮ピラー4本の画像案

| ページ | 画像テーマ |
| --- | --- |
| 実家の片付け | 親の家で荷物を3分類している親子 |
| 売れるものの整理 | 段ボールに古い本、カメラ、レコード、ブランド品が並ぶ |
| 買取・処分の進め方 | 買取、処分、確認中の箱に荷物を分ける |
| 家族の悩み | 親子が古い物を見ながら落ち着いて話し合う |

## 画像サイズ

WordPressのアイキャッチは、まず以下を基準にします。

- 横長: 1200 x 630
- 記事内図解: 1200 x 800
- SNS流用を考える場合: 1200 x 1200

## 使い回し方

最初は各ピラー1枚ずつで十分です。

その後、品目別記事が増えてきたら、ジャンルごとに画像を追加します。

- 本、専門書
- カメラ、レンズ
- レコード、CD
- ブランド品
- 切手、古銭
- フィギュア、おもちゃ
- 工具、楽器
