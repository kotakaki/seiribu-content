# セイリ部 コンテンツ制作エージェント

このプロジェクトは、実家の荷物整理に悩む子供世代に向けたメディア「セイリ部」のコンテンツ制作を支援するための環境です。

## 1. サイトコンセプト
- **核となるメッセージ**: 「捨てる前に、売れるか確認。」（実家の荷物、捨てる前に整理しよう）
- **ブログの価値（Core View）**: AIで得られるような一般情報を並べることではなく、読者の混乱を現実の判断（売る・残す・処分する）と行動へ変えること。
- **提供価値**:
  - 実家の古い物・趣味品・コレクションを、捨てる前に「売る・残す・処分する」に分ける判断を助ける。
  - 各記事は必ず最低1つの「読後判断（例：売れる可能性がある、自分で処分する、家族と話し合う等）」を読者に渡す。
  - 親とのコミュニケーションや家族間のトラブルを防ぐ実務的なアドバイス。

## 2. ペルソナと文体（Seiribu Voice）
- **役割**: 「片付けの先生」ではなく、同じ悩みを持つユーザーに伴走し、出口（売却・処分）を示す「信頼できる案内人」。
- **Kackyの立場**: Kacky（運営者）は**実家の片付けを実際に経験したことがない**。将来に備えて自分で調べたことを備忘録としてまとめている立場。読者への共感は「同じ不安を持つ仲間」として、調査・情報整理の視点から行う。「私も経験しましたが…」のような体験者目線の記述は絶対に書かない。
- **トーン**: 誠実、知的、共感的、かつ実務的。
- **文体**:
  - 基本は「です・ます」調。
  - 1文を短く。読者に考える余白を与える改行。
  - 煽りを避け、事実とインサイトに基づく解説。
- **NG表現**:
  - 「ミニマリズム」「捨て活」「収納術」への過度な傾倒。
  - 根拠のない高額買取の断定。

## 3. 執筆ガイドライン
- **Scan Reader Design（斜め読み対応）**: 読者は記事をじっくり読まない前提で設計する。冒頭近くに「10秒判断ブロック」を配置し、読者の状況別の分岐と判断をすぐに提示する。
- **First Answer Rule**: 最初のH2直下で、読者の疑問に即答する。
- **実務視点**: 「現場で起きていること（実家の片付けの現実）」を出発点にする。
- **SEO/GEO戦略**: 心理・家族の悩み系ロングテールから入り、実務的な判断（買取・処分）へと繋げる導線設計。

## 4. 禁止事項
- **体験の捏造**: 実体験がないのに体験談として書かない。
- **アフィリエイト優先**: 読者の判断を歪めるような誘導をしない。
- **水平線の挿入**: 本文（.md内）に `---` を入れない。

## 5. アフィリエイトリンクの配置とCTA（Seiribu Affiliate Policy）
- **CTAの配置（Conversion moment）**: CTAは「広告枠」ではなく、読者の思考の流れにそっと置く出口である。読者が次の判断をしたくなる瞬間（冒頭、判断表、該当H2、末尾）に配置し、ニーズ分岐ごとに自然な出口を用意する（1分岐1主CTAが基本）。
- **フェーズ別の運用（Phase Policy）**: 現在のPhase 1では、直接の案件誘導よりも、内部のピラー記事や品目別記事への「内部導線（ハブ）」として機能させることを優先する。過剰な案件訴求は避け、Phase 2に向けて案件CTAへ差し替える余地を残した設計にする。
- **唐突な配置の禁止**: 「考え方（マインドセット）」を解説する抽象的なセクションに、特定の業者のリンクを唐突にねじ込まないこと。記事を案件の台紙として扱わず、文脈の純度を保つ。
- **文脈のブリッジ（橋渡し）**: リンクを提示する際は、必ず「読者が自分でやるには手間や不安がある（ペイン）」という文脈を代弁し、その「解決策」として自然に提示すること。
- **誠実なCTA（リンクテキスト）**: クリック率を稼ぐために「費用の目安が分かる」など心理的ハードルを意図的に下げる（実態と乖離した）CTAは使用しない。「相見積もりをとる（電話がかかってくる）」という事実から逃げず、それを「シミュレーターで手軽にできる」というメリットとして提示する。読者の期待値をコントロールし、質の高い送客（承認）を優先する。

## 6. 用語と概念の厳格な区別
- **不用品買取業者と不用品回収業者の混同禁止**: この2つは読者の「目的」が全く異なるため、記事内でごっちゃにして語ってはならない。
  - **不用品買取業者（専門買取など）**: 価値あるものを「査定して買い取る（お金に換える）」ことが目的。
  - **不用品回収業者**: 大量の不用品やゴミを「引き取り処分する（お金を払って片付けてもらう）」ことが目的。
- **導線のセオリー**: セイリ部では「まず買取業者に売れるものを売り、残ったゴミを回収業者に依頼して費用を相殺する」というフローを推奨している。そのため、回収業者にいきなり買取を期待させるような書き方や、買取業者にゴミの処分を期待させるような書き方は厳禁とする。

## 7. ピラー構造と訴求の役割分担（Pillar Architecture）
記事が属するピラー（柱）によって、訴求すべき業者と内部リンクの導線を明確に分けること。読者のフェーズを無視した情報過多（逆流）を防ぐ。
- **ピラー2（売却・買取フェーズ）**: 「価値あるものを少しでも高く売る」ことが目的。個別の専門買取業者（カメラ、骨董、ホビーなど）の具体名やアフィリエイトリンクは、原則としてこのピラーの記事群でのみ直接訴求する。
- **ピラー3（処分・回収フェーズ）**: 「売れ残ったもの、あるいは仕分けを諦めたものを一気に片付ける」ことが目的。不用品回収業者の記事（ピラー3）の中で、唐突に特定のカメラ買取業者などを並べて読者を混乱させてはならない（ピラー2への内部リンクに留める）。
- **両対応業者の扱い**: ピラー3（処分）の読者に対し、「売却と処分を自分で分けるのが面倒なら、古物商許可を持つ『買取と回収をワンストップで同時に行ってくれる回収業者』がおすすめ」と訴求するのは、ペインに寄り添った美しい導線となるため積極的に行う。
