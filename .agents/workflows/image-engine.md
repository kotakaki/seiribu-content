# セイリ部 画像制作エンジン

完成済みの記事Markdownを入力にして、AI画像生成プロンプトの構築、および画像制作プランの生成を行うワークフローです。

## 起動方法

```bash
python3 seiribu-editorial/scripts/generate_image_plan.py \
  seiribu-editorial/drafts/pillars/sell-or-dispose-clean.md
```

## 【AIへの絶対遵守ルール：文字・ロゴの描画禁止】
画像生成AIを使用する場合、以下の制約を絶対に守ること。
- 生成する素材画像内に、**絶対に文字（日本語・英語・記号）、ロゴ、説明ラベルなどを描画してはいけません**。
- あなたのタスクは「純粋なビジュアル素材（背景・人物・物）」を作ることのみです。見出しやロゴは、後工程で人間（Canva等）や別スクリプトが追加します。
- したがって「文字パネル」や「白カード」などの余白も無理に作る必要はありません。

## 【基本方針とトーン】
- **画風（アートスタイル）**: 温かみのあるフラットイラスト（主線なし）、または日本の現代的な日常を表現した高品質なベクターアート。写真風にする場合は、生々しすぎない明るいトーン。
- **トーン**: やさしい、清潔、少し生活感がある。不安を煽らない。50代以上にも親しみやすい。
- **テーマ**: おしゃれな収納写真より、実家の荷物、段ボール、古い物、確認や仕分けの状況が伝わる画像。読者が「これは自分の状況に近い」と感じることを優先する。
- **図解と情景のバランス**: B2B向けのコンサル資料ではないため、図解（比較表やフロー）だけでなく、雰囲気を伝える情景イラスト（親子で話し合う姿など）もバランスよく配置する。

## 【避けたい画像（NG表現）】
- 真っ白で生活感がないミニマリスト部屋
- 高級感だけが強い買取業者風の画像
- 汚部屋、ゴミ屋敷感が強すぎる画像
- 暗い遺品整理、孤独、恐怖を煽る画像
- 現金や高額査定を強調する画像

## 【画像サイズ（アスペクト比）と制作物の状態管理】

画像生成ツールを呼び出す際は、ピクセルではなく以下の「アスペクト比」を指定して出力すること。
- **アイキャッチ用素材**: `16:9` または `3:2`（横長）
- **記事内図解用素材**: `3:2`（横長）
- **SNS流用前提の素材**: `1:1`（正方形）

生成した画像は、以下のステータスで厳格に管理・保存する。
- `source/`: 生成直後や背景透過前の元素材
- `draft/`: 採用前の候補
- `final/`: 記事やCanvaで使う確定画像
- `rejected/`: 不採用画像（公開用の `assets/images/` 等には絶対に入れない）

## 【アイキャッチのブランド型とロゴ配置（後工程用）】
※AIは画像生成時にこれらを描画しない。最終組み立て時の仕様。
- **Canvaテンプレ**: `seiribu-editorial/config/image-engine.json` の `canva_template_url` (https://canva.link/mqlqak3adj01g1i) を使用。
- **構成**: ロゴは左上、見出しは上部中央に大きく、サブタイトルは見出し直下。ビジュアル素材は下部から中央へ配置。
- **フォント**: 見出しはUDモトヤアポロ太字、サブタイトルはNoto Sans JP Regular。
- **ロゴ元画像**: `assets/images/brand/seiribu-logo.png`
- **注意点**: 見出し、図解ラベル、人物の顔、重要な品物の上にロゴを重ねない。画像全体の主役は記事内容にし、ロゴは小さなブランド署名として扱う。

## 【展開・派生の考え方】
- 同じ人物配置、同じ箱分け、同じ背景を別記事へ貼り替えるだけの流用は禁止。
- 同じブランドトーンを保ちながら、記事固有の主役（本、カメラ、着物など）に合わせて描き分けること。
